# Product Overview

**Bit Brainic** is an AI-powered Computer Science tutoring platform that provides personalized learning experiences for programming and CS concepts.

## Core Features
- AI-powered chat tutoring using Google's Gemini API
- User authentication and session management
- Admin dashboard for user management
- Contact system for user inquiries
- Responsive web interface with modern UI/UX

## Target Users
- Computer science students
- Programming learners
- Developers seeking CS concept clarification

## Key Value Proposition
Provides instant, personalized AI guidance for learning programming, algorithms, and computer science concepts through an interactive chat interface.
